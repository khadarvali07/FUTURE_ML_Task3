# AI Powered Resume Screening System

### Intelligent Candidate Shortlisting using NLP and Machine Learning

## Project Overview

The AI Powered Resume Screening System is a Machine Learning and Natural Language Processing based recruitment automation project designed to simplify the hiring process by automatically evaluating candidate resumes against specific job requirements.

Traditional recruitment processes require manual screening of hundreds of resumes, which is time-consuming and often inefficient. This project solves that problem by building an intelligent system that extracts candidate skills, analyzes resume content, compares it with a job description, calculates similarity scores, and ranks candidates automatically.

The system helps recruiters identify the most suitable candidates while reducing human effort and improving hiring efficiency.

---

## Problem Statement

Organizations receive a large number of resumes for every job opening. Manually checking each resume consumes significant time and may lead to biased or inconsistent selection.

This project aims to automate candidate screening by developing an intelligent system capable of understanding resume content and matching it with required job skills.

---

## Objectives

* Automate the resume shortlisting process
* Extract relevant skills and keywords from candidate resumes
* Compare candidate profiles with job descriptions
* Rank candidates based on similarity score
* Identify missing skills for each candidate
* Improve recruitment speed and decision making
